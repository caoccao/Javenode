=====
Javet
=====

The ``javet`` module provides some internal Javet and JVM API.

API
===

* ``package``
   * ``.getPackages``
   * ``.implementationTitle``
   * ``.implementationVersion``
   * ``.implementationVendor``
   * ``.name``
   * ``.sealed``
   * ``.specificationTitle``
   * ``.specificationVersion``
   * ``.specificationVendor``
   * ``.valid``
* ``v8``
   * ``gc()``
